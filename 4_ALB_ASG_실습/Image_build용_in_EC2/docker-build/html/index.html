Docker 설치 및 설정

다양한 운영체제 환경에서의 초기 구축 가이드

Dockerfile 지시어

  • FROM: 베이스 이미지 지정
  • WORKDIR: 작업 디렉토리 설정
  • COPY/ADD: 파일 복사
  • RUN: 빌드 시 실행할 명령 (패키지 설치)
  • CMD: 컨테이너 시작 시 명령

주요 환경별 설정 이미지

Nginx FastAPI (Python) Apache (HTTPD) MySQL MongoDB DynamoDB Local

* 각 이미지별 최적화된 Dockerfile 작성이 중요합니다.
