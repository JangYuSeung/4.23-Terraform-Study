Docker 설치 및 설정

다양한 운영체제 환경에서의 초기 구축 가이드

핵심 명령어 가이드

명령어 설명 핵심 옵션
docker run 컨테이너 생성 및 실행 -d (백그라운드), -p (포트), -v (볼륨)
docker exec 실행 중인 컨테이너 명령 전달 -it (대화형 터미널)
docker network 컨테이너 간 네트워크 설정 ls, create, connect
docker volume 데이터 영속성 관리 create, inspect

실습 예제: Nginx & FastAPI

1. Nginx 웹서버
docker run -d --name my-web -p 80:80 nginx
2. FastAPI 앱
docker run -d --name my-api -p 8000:80 tiangolo/uvicorn-gunicorn-fastapi
